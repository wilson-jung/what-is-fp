### 함수 조합으로 함수 만들기

#### 중복이 존재하는 다음 코드를 재작성 해보자

```js
const products = [
  { name: "a", price: 1000 },
  { name: "b", price: 1100 },
  { name: "c", price: 1010 },
  { name: "d", price: 1001 },
];

go(
  products,
  filter((p) => p.price < 1010),
  map((p) => p.price),
  reduce(add),
  console.log
);

go(
  products,
  filter((p) => p.price >= 1010),
  map((p) => p.price),
  reduce(add),
  console.log
);
```
#### 중복을 제거 해보자
```js
const totalPrice = pipe(
  map((p) => p.price),
  reduce(add)
);
go(
  products,
  filter((p) => p.price < 1010),
  totalPrice,
  console.log
);

go(
  products,
  filter((p) => p.price >= 1010),
  totalPrice,
  console.log
);
```
